Migrate to Merchandising from Other Software

Looking to migrate from another billing software to Merchandising? It’s now made easy as ever! Just go according to the steps below and you’re done.

Step 1: Register in Merchandising

  1. Go to http://merchandising.mobiato-msfa.com/auth/signup
  2. Fill the first name, last name, email, phone no, password.
  3. Select Country from drop down list.
  4. Check I agree to terms of service and privacy policy.
  5. Click signup button.

You can also signup with your google account.

Step 2: Signin

  1. After registeration go to http://merchandising.mobiato-msfa.com/auth/login
  2. Enter your user name or email.
  3. Enter your password.
  4. Click on login button to sign in.
  5. You can also login with google account.

Step 3: Creating Organization Profile

Get your organization registered with merchandising with the following simple steps:

  1. Go to the top right corner of the screen and click the settings image.
  2. Select Organization Setting from the list.
  3. Fill the details of the organization.
    • Add the logo of size (Preferred Image Size: 240px x 240px@72 DPI Maximum size of 1 MB) of the organization which would be displayed on the estimates, invoices, etc.
    • Provide the Organization name.
    • Business location.
    • Company Address
    • Contact person
    • Provide with the currency used in the Organization
    • Select the fiscal year used in organization.
    • Add TaxID.
    • Can add Additional field if required.
    • Click save button to add Organization details.

Now the Profile is created and other services of Merchandising are available.

Note:’*’ is used for mandatory fields.

Step 4: Assign User Roles

You can assign roles to the users here and can see all users and the roles assigned to them using two tabs ALL USERS and ROLES.

  1. Go to the right left corner of the screen and click INVITE USER button.
  2. Fill first name, last name, email address, phone number of the user.
  3. Select roles from the drop down.
  4. Click the INVITE USER button to save the role else click CANCEL button.
  5. In ALL USER tab the list of users and roles are listed.
  6. Roles tab shows all the roles created and description of the same.

To add new role to the user, go to the top right corner of the screen and click on the NEW ROLE button. Fill the Role name, Description and check the access limits as per the requirement and click save button to save the role.

You can make use of copy, edit or delete options given in ALL USERS tab.

Step 5: Enable the required modules

Preferences under settings allows to enables the modules.

  1. Under preferences Costume modules are listed and can be checked.
  2. Add the work flow rules.
  3. Templates can be used for modules.
  4. Activity profiles is used for maintaining the duration for each activity.
  5. Themes change option allows to set different themes to the pages.

Step 6: Applying Tax

Taxes play a vital role in the finance sector of any Organization and each country has their own way of Tax implementation, so merchandising helps manage this variation.

  1. Go to the top right corner and click on settings image and open Taxes option
  2. Click on the +New for adding tax name and rate.
  3. Click save button to save the new tax.
  4. Setting for GST is also present.

Step 8: Adding Bank Account

Add Bank details for the transactions purpose.

  1. Go to the top right corner and click on settings image and open Bank option
  2. Click on the +New for adding tax.
  3. Provide bank code, bank name, account number and address
  4. Click save button to save the bank details.
  5. The new account will be shown in the list.

Step 9: Adding Currencies

Your Organization might be dealing with the customers from different countries, you can add those currencies in Merchandising

  1. Go to top left corner of the screen and click the settings image and select currency option.
  2. Click the +New button on the top right corner of the screen.
  3. Fill the form by providing currency code, currency symbol, currency name, decimal places and format.
  4. Click save button to save the new currency.
  5. The newly added currency will be displayed in the list of currencies.
  6. You can enable Exchange rate feeds using the button provided.
  7. You can use the options for edit, view or delete the exchange rates by touching the list item.

Step 10: Location

Country
  1. Go to the top right corner of the screen and click the +New button to add a new country for the warehouse.
  2. Provide country code, name, dial code, currency, currency code, currency symbol.
  3. Click the save button to add the country.
  4. Newly added country is displayed in the list.

Country code can be added manually by clicking the settings image in front of country code and checking “I will add them manually each time option.” And saving the choice by clicking save button.

When the country is selected from the list you can find few more options:
